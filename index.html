Microsoft foundation + coding extensions

Give every sentence a job.

Start with the Goal, Context, Source, and Expectations components described in Microsoft’s Copilot prompt guidance. A clear goal is the minimum; use the other components when they improve the request. For coding work, this lab adds Constraints and Verification so scope and completion are explicit.

Inspect the coach

Nothing hidden behind the score.

The coach uses repeatable browser-side rules. It cannot understand your repository or guarantee that an answer will be correct.

Where this framework comes from

Microsoft’s Copilot prompt guidance describes four useful components: Goal, Context, Source, and Expectations, while noting that only a clear goal is required. GitHub Copilot best practices additionally recommend relevant context, examples, specific requirements, smaller tasks, iteration, and validation. Prompt Guide labels Constraints and Verification as its own coding extensions.

What the coach can detect

Visible language signals: action verbs, audience or situation, named files and error evidence, constraints, testable completion criteria, requested formats, repetition, filler, and conflicting phrases.

What it cannot judge

Whether context is factually correct, whether a named file is truly relevant, whether the model will succeed, or whether a technical solution is safe. Treat every finding as a review prompt, verify generated work with appropriate tests and human judgment, and apply additional review for consequential uses.

How model pricing comparison works

The page uses a simple, provider-independent word-and-punctuation heuristic for prompt tokens, then applies a dated snapshot of official Azure OpenAI Global Standard and Vertex AI Global text API rates. Actual tokenization varies by model, content, and provider; use provider-reported usage for billing reconciliation. A model name alone is not a billable SKU, so every row names its hosting platform and deployment basis and links to its official source. Cache schemes that require storage or write charges are not ranked against simple discounted-input caching.

What pricing is excluded

GitHub Copilot, ChatGPT, and Claude subscriptions use licenses or credits rather than these API token rates. The estimate also excludes enterprise agreements, taxes, currency conversion, batch, priority, regional, provisioned, long-context, media, tools, grounding, hidden instructions, retrieved history, retries, and provider rounding. For reasoning models, billed output can include hidden reasoning tokens; enter total expected billable output because this page does not predict that hidden-token volume.
